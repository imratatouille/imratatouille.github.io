1. < h > 태그

h 태그 사용법: < h1 > 내용 < /h1 >

h1 hello world

h2 hello world

h3 hello world

h4 hello world

h5 hello world
h6 hello world

2. < p > 태그

p 태그 사용법: < p > 내용 < /p >

hello world

3. < a > 태그

a 태그 사용법: < a href="주소" > 표시될 내용 < /a >

this is a tag

4. < b >, < strong > 태그

b 태그 사용법: < b > 내용 < /b >

strong 태그 사용법: < strong > 내용 < /strong >

b tag : hello world strong tag : hello world

5. < i >, < em > 태그


i 태그 사용법: < i > 내용 < /i >

em 태그 사용법: < em > 내용 < /em >

i tag : hello world em tag: hello world

6. < ul >, < ol >, < li > 태그

ul 태그 사용법:
< ul >
    < li > 내용 < /li >
< /ul >

ol 태그 사용법:
< ol >
    < li > 내용 < /li >
< /ol >

  • ul => 순서 없는 목록
  • li => 목록 항목을 정의
  1. ol => 순서 있는 목록
  2. li => 목록 항목을 정의

7. < img > 태그


img 태그 사용법: < img src="이미지경로" >

8. < input >, < button > 태그

input 태그 사용법: < input >

button 태그 사용법: < button > 내용 < /button >

9. < !-- -- >주석


사용법: < !-- 내용 -- >

여기서 주석을 볼려면 개발자 도구(f12)를 눌러서 확인을 해주세요
